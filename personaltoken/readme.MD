# Personatoken Standard v1.1.2
Ghost Router – AI-Native Zero-Trust Identity Layer

This folder contains the complete standard for the Personatoken identity
framework. Personatoken defines dynamic, context-aware identities for
AI agents. It includes permissions, tools, constraints, telemetry,
trust, delegation, shadow/honeypot personas, quarantine forks and riot-mode.

## Files

- **personatoken-v1.1.2.schema.json** – Formal JSON Schema
- **examples/** – Valid example tokens
- **trust-engine-v4.md** – Reference trust-score logic

## Features in v1.1.2
- Dynamic persona identity
- Multi-mode token types: full, light, delegated, shadow, fork, riot
- Honeypot and shadow personas
- Quarantine fork identities
- Self-destructing tokens for sandboxing
- Riot-mode cluster lockdown
- Capability- and constraint-based tools
- Telemetry-driven trust
- Cryptographic binding (device/IP/session)

This is the world's first AI-native Zero-Trust agent identity spec.
